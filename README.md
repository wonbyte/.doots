## .doots

linux dotfiles managed with [GNU Stow](https://www.gnu.org/software/stow/).

### Install

```sh
git clone https://github.com/wonbyte/.doots ~/.doots
cd ~/.doots
./install.sh
```

### Structure

| Directory | Config |
|-----------|--------|
| `bin` | Scripts (`~/.local/bin`) |
| `fish` | Fish shell |
| `nvim` | Neovim |
| `starship` | Starship prompt |
| `tmux` | tmux |
